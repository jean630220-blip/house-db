💡 如何取得 Google API 金鑰？（只需設定一次）

  1. 前往 Google Cloud Console 建立一個新專案。
  2. 在「API 與服務」中啟用 **Google Drive API**。
  3. 在「OAuth 同意畫面」設定為 External (外部)，填寫必要資訊並送出。
  4. 在「憑證」中：
    • 建立 **API 金鑰 (API Key)**。
    • 建立 **OAuth 用戶端 ID (Client ID)**，應用程式類型選擇「網頁應用程式」，並在 **已授權的 JavaScript 來源** 加入您的部署網址（例如：`https://username.github.io` 與 `http://localhost` 或 `http://127.0.0.1`）。
  5. 將產生的 Client ID 與 API Key 貼到上方，點擊儲存並完成登入授權即可！
